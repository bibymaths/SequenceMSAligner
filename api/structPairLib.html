Detailed Description

Stores all optimal pairwise residue-pair positions for an input set.

For every ordered pair (i, k) where i < k, lib[i][k] contains the set of (ungapped_pos_i, ungapped_pos_k) integer pairs that co-occur in the optimal global alignment of sequences i and k. This library is the foundation of the COFFEE Log-Expectation scoring used by calculate_le_score() and build_profile_le().
